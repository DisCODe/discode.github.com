Move one of the components to another thread

Description

Modify the TriggeredSequenceViewer.xml in order to add the displayed image refreshment.

Task structure

The structure of the RefreshedTriggeredSequenceViewer task is shown below.

Note:
For more details regarding the structure please click here.

Task file

Create a new RefreshedTriggeredSequenceViewer.xml file on the base of the previously created TriggeredSequenceViewer.xml. Modify required parts:

  • Move the Wnd1 component to a separate Exec3 executor. 
        <Components>
        ...
        <Wnd1 type="CvWindow" thread="Exec3" group="s1">
            <title>Preview</title>
        </Wnd1>
        ...
  • Create another executor and set the main thread component to Wnd1. 
        <Executors>
        ...
        <Exec3 type="periodic">
            <main_component>Wnd1</main_component>
            <interval>0.2</interval>
        </Exec1>
        ...
    Note:
    Note that the new executor is "periodic", thus require an additional parameter (interval, in seconds) to be set.

Result

A sequence of images will be displayed. After an image will be shown application will wait for user to enter the 't' (must be followed by <enter>) to load and display another. Besides the displayed images will be refreshed every 0.2s.
