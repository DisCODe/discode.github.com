Detailed Description

template<typename T, template< class T > class BufferingPolicy = DataStreamBuffer::Queue, class Sync = Synchronization::NoSync>
class Base::DataStreamInPtr< T, BufferingPolicy, Sync >

Input data stream, retrieve pointer to stored data.

It's usefull when one have to receive pointer to base class, when objects of derived class are sent.

Template Parameters:
T type of data to be handled by DataStream
BufferingPolicy buffering policy (way to store data)
ReadSync synchronization of read access
WriteSync synchronization of write access
