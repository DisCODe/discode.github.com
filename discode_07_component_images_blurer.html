Tutorial objectives

The goal is to create a component responsible for bluring of the images.

Utilize the standard mean filter with 3-by-3 kernel shown below.

1 1 1
1 1 1
1 1 1

Instead of implementing the blue operation on your own please use the cv::blur function from the OpenCV.

The ImageBlurer component

Utilize the existing tools to create the component body, configure cmake and generate makefile.

Select your operating system:

New Component in SDK Describes how to utilize the SDK in order to create and compile new component under Ubuntu OS.
New Component in SDK Describes how to utilize the SDK in order to create and compile new component under Windows OS (32bit).

Add OpenCV library to list of linked libraries.

Edit the CMakeLists.txt file in order to uncomment the OpenCV library dependency. 

...
FIND_PACKAGE( OpenCV REQUIRED )
SET(LIBS ${OpenCV_LIBS} ${OpenCV_LIBRARIES} ${LIBS} )
...
Note:
To uncomment given line remove the # character from the front.

Edit the header file ImageBlurer.hpp

Edit the component header and perform changes listed below.

  • Add required OpenCV headers after the existing ones. 
    ...
#include <cv.h>
...
  • Add access to the OpenCV namespace cv. 
    ...
using namespace cv;
...
  • Enter the protected section of the ImageBlurer_Processor class and add input and output data streams. 
        ...
    Base::DataStreamIn <Mat> in_img;

    Base::DataStreamOut <Mat> out_img;
    ...
  • Add event handler (when a new image will arrived) and event (raised when image the image processing will be done). 
        ...
    void onNewImage();

    Base::EventHandler <ImageBlurer_Processor> h_onNewImage;

    Base::Event * newImage;
    ...

Edit the source file ImageBlurer.cpp

Edit the component source and perform changes listed below.

  • In the body of the OnInit() method register data streams, events and event handlers. 
        ...
    // Register handler.
    h_onNewImage.setup(this, &ImageBlurer_Processor::onNewImage);
    registerHandler("onNewImage", &h_onNewImage);
    // Register event.
    newImage = registerEvent("newImage");

    // Register data streams.
    registerStream("in_img", &in_img);
    registerStream("out_img", &out_img);
    ...
  • Add new onNewImage() method. 
    ...
void ImageBlurer_Processor::onNewImage()
{
    LOG(LTRACE) << "ImageBlurer_Processor::onNewImage\n";
    try {
        // Read image from input data stream.
        Mat img = in_img.read();
        // Perform bluring.
        blur(img, img, cv::Size(10,10));
        // Write blured image to output data stream.
        out_img.write(img);
        // Raise event.
        newImage->raise();
    } catch (...) {
        LOG(LERROR) << "ImageBlurer_Processor::onNewImage failed\n";
    }
}
...
  • Compile and copy the component to bin library.

Task file

Create a new ImageBlurer.xml file on the base of the RefreshedTriggeredSequenceViewer.xml.

  • Add the Blur1 component to the list of components, set its executor to Exec1 
            ...
        <Blur1 type="ImageBlurer" thread="Exec1" group="s1">
        </Blur1>
        ...
  • Connect data ports between: Seq1.Out - Blur1.In and Blur1.Out - Wnd1.In. 
            ...
        <Seq1>
            <out_img type="out" group="Con1"/>
        </Seq1>
        <Blur1>
            <in_img  type="in" group="Con1"/>
        </Blur1>

        <Blur1>
            <out_img type="out" group="Con2"/>
        </Blur1>
        <Wnd1>
            <in_img  type="in" group="Con2"/>
        </Wnd1>
        ...
  • Connect events and event handlers between: Seq1.NewImage - Blur1.onNewImage and Blur1.NewImage - Wnd1.onNewImage. 
            ...
        <Event source="Seq1.newImage" destination="Blur1.onNewImage"/>
        <Event source="Blur1.newImage" destination="Wnd1.onNewImage"/>
        ...
Note:
For more task file details please refer to the previous tutorials.
