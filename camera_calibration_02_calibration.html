Perform the calibration computations

Tutorial objectives

The aim is to create a task responsible for computation of the calibration parameters (camera intrinsics and distortion vector).

Task structure

The task consists of five components. Besides the implementation of new component the functionality of existing ones should be expanded.

Note:
For more details regarding the structure please click here.

The Keyboard component

The component should emit two events, related to two different keys:

  • the 't' key should emit an event handled by the OnTrigger handler of the Seq1 component,
  • the 'c' key should raise an event handled by the OnCalibration handler of the Clbr1 component.

The ChessboardLocator component

The Chess1 should be expanded by new output data port and raised event.

When a chessboard is properly detected a structure containing data related to its corners should be send through the Outboard out data stream and a NewBoard event should be emited.

Note:
Think about the form of the structure - it should be the template parameter of both Outboard and Inboard data streams.

The CameraCalibration component

The component is responsible for storing data delivered by the Chess1 component. After the last image from the calibration set is processed the user should press the 'c' key in order to send a command to Clbr1 component that it should perform the calibration computations.

Hints

  • Utilize the OpenCV calibrateCamera() function to perform calibration computations.
  • Save computed intrinsic matrix and distortion coefficients to files.

Result

Calibration parameters will be computed.
