CvMorphology
[Processors]

Performs advanced morphological transformations

Data streams:
  • [in]  in_img (cv::Mat)
    Input image
  • [out] out_img (cv::Mat)
    Output image
Events:
  • newImage
    New image is ready
Event handlers:
  • onNewImage
    New image arrived
Properties:
  • type (string="MORPH_OPEN")
    Type of morphological operation, one of the following:
    • MORPH_OPEN opening
    • MORPH_CLOSE closing
    • MORPH_GRADIENT morphological gradient
    • MORPH_TOPHAT “top hat”
    • MORPH_BLACKHAT “black hat”
  • iterations (int=1)
    Number of times erosion and dilation are applied
