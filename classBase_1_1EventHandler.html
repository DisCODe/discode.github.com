Detailed Description

template<class Class>
class Base::EventHandler< Class >

EventHandler is called when associated Event is raised.

EventHandler calls it's owners method, which must be of type Method (taking no arguments and returning void). If EventHandler is connected to Event, but has no associated owner and method then exception is thrown when event is raised.
