Detailed Description

template<class T>
class Base::Singleton< T >

Template singleton class, implemented in a Boost-singleton manner.

Object is stored in scoped_ptr and instance initialization is protected by Boost::call_once. Warning: If T's constructor throws, instance() will return a null reference.
