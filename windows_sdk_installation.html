Windows SDK

↑ Installation instructions

There are two elements that must be installed:

  • the DisCODe developement environment, containing tools and libraries required during the component developement and compilation,
  • the DisCODe SDK for Windows.

The DisCODe environment

  • Download the latest version of environment from here: DisCODeEnv.exe.
  • Install the environment by simply running the DisCODeEnv.exe and enter the destination path (e.g. "D:\" will extract the environment to the "D:\DisCODeEnv\". The output directory will be called env_dest_path).

DisCODe SDK

  • Download the latest version of SDK from here: DisCODe-1.0.0-win32.exe.
  • Run the SDK installator. Read and accept the license agreement.
  • Set the destination path sdk_dest_path (for further cmake configuration it is suggested to install the SDK near (or inside) the env_dest_path directory, e.g. "D:\DisCODeEnv\DisCODe").
