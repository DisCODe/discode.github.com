CvPixelOperator
[Processors]

Per pixel operations on image.

This processor allows to do bitwise operations with constant, non-linear functions etc.

Data streams:
  • [in]  in_img (cv::Mat)
    Input image
  • [out] out_img (cv::Mat)
    Output image
Events:
  • newImage
    New image is ready
Event handlers:
  • onNewImage
    New image arrived
Properties:
  • operator (string="unit")
    Operator to be used. Possible values (with corresponding settings) are listed below.
Possible operators:
  • unit - identity operator:
  • tanh - hiperbolic tangent
    • sigma (double=0.1)
    • mean (double=0.0)
  • and - bitwise conjunction
    • mask (int=0xFF)
  • or - bitwise alternative
    • mask (int=0xFF)
