Tutorial objectives

The goal is to display a calibration sequence after performing the undistortion operation with the use of calibration parameters computed in the previous tutorial.

Task structure

Note:
For more details regarding the structure please click here.

The ImageUndistortion component

The component should perform the undistortion operation on received image and pass it to output data stream.

Hints

  • Load intrinsic matrix and distortion coefficients (computed during the calibration task) from previously created files.
  • Use the OpenCV undistort() function.

Result

A set of undistorted images will be displayed.
